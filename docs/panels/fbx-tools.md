# FBX Tools (removed)

!!! note "This panel was removed"
    The standalone **FBX Tools** panel no longer exists. Its only remaining feature — the **Better FBX** status — now appears directly in the **[Export](export.md)** panel (when the FBX target is selected). The old **Import FBX** button was dropped: use Blender's native **File → Import → FBX** instead.

If you have the [Better FBX Importer & Exporter](https://blendermarket.com/products/better-fbx-importer--exporter) addon installed, the Export panel detects it and routes FBX export through its dialog automatically.
